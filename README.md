## 🚀 실행 방법

### 1. Node.js 버전 설정

- **권장 버전:** Node.js `22.14.0`

```bash
# nvm 사용 시
nvm install           # 프로젝트 권장 버전 설치
nvm use 22.14.0               # 권장 버전으로 전환

#### 프로젝트 실행방법
```bash
npm install   # .nvmrc 파일 기준 권장 Node.js 버전 설치
npm run dev       # 해당 프로젝트 디렉토리에서 권장 버전 적용
